# Beer Club Demo App

## To run the app: 

1. Install packages
`npm install`

2. Start the json server (our mock server)
`npm run server`

3. Start the webapp
`npm start`

## Available Scripts

This project relies on:
- npm
- json-server
- bootstrap

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
